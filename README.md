# 🌐 Developer Portfolio

A modern, responsive **developer portfolio website** built with **HTML, JavaScript, and Tailwind CSS**. Designed to showcase technical skills, projects, experience, and achievements through a clean, high-performance interface.

---

## 🚀 Features

* 🎨 Modern Bento Grid layout for a clean and professional design.
* 📱 Fully responsive across desktop, tablet, and mobile devices.
* 💼 Dedicated sections for projects, skills, experience, and education.
* 📬 Interactive contact form powered by **Web3Forms** for direct email submissions.
* ✨ Smooth animations, transitions, and interactive UI elements.
* ⚡ Fast-loading and optimized for performance.
* 🎯 Clean, well-structured, and easy-to-customize codebase.

---

* HTML5

## ⚙️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/your-portfolio.git
cd your-portfolio
```

### 2. Open the Project

Open `index.html` in your preferred code editor or run it using **VS Code Live Server**.

### 3. Configure the Contact Form

Create a free account on **Web3Forms** and generate an **Access Key**.

Replace the placeholder in your contact form:

```html
<input
  type="hidden"
  name="access_key"
  value="YOUR_WEB3FORMS_ACCESS_KEY"
/>
```

Your contact form will now send submissions directly to your email.
